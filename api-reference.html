ShopFlow E-commerce API (1.0.0)

Download OpenAPI specification:Download

API Support: support@shopflow.com

APIs to build online stores, manage products, handle orders, and process payments.

Products

All product management operations

Get all products

Returns a paginated list of all products in ShopFlow.

Authorizations:
apiKeyAuth
query Parameters
category
string
Enum: "Electronics" "Clothing" "Books" "Home" "Sports"

Filter products by category

min_price
number

Filter products with price greater than or equal to this value

max_price
number

Filter products with price less than or equal to this value

sort
string
Default: "newest"
Enum: "price_asc" "price_desc" "newest" "popular"

Sort products by selected field

in_stock
boolean
Default: true

Filter only products currently in stock

limit
integer
Default: 20

Maximum number of products to return per page

Responses

Response Schema: application/json
Array
id
integer

Unique identifier for the product

name
string

Name of the product

description
string

Detailed description of the product

price
number <float>

Price of the product

currency
string

Currency of the price

category
string

Product category

stock_quantity
integer

Number of units available in stock

sku
string

Unique product identifier for inventory

discount_percentage
number

Discount percentage applied to the product

created_at
string <date-time>

Timestamp when the product was created

Production server

https://api.shopflow.com/v1/products

Sandbox for testing

https://sandbox.shopflow.com/v1/products
